## Short Chinese version

Launch Desk Base Agent 是一個 Base-native AI workflow agent demo。

它測試的是：AI 不只是聊天，而是可以擁有受控的鏈上操作身份，執行 launch / release workflow，並把重要結果留下可驗證的 proof。

目前已完成：

- CDP Server Wallet 作為 AI operator wallet
- Base mainnet smart contract proof
- x402 / USDC paid API unlock 概念
- Founder wallet 收款路由
- Public API catalog 和 proof endpoint

收入設計上，早期不讓 AI operator 持有大量資金。付費 API 收入先直接進 founder wallet，AI operator 只保留少量 gas，用來執行經授權的 proof actions。

## Short English version

Launch Desk Base Agent is a Base-native AI workflow agent demo.

It explores how an AI system can move beyond chat into controlled onchain workflow execution for launch and release operations.

Current proof includes:

- CDP Server Wallet as the AI operator wallet
- Base mainnet smart contract proof
- x402 / USDC paid API unlock concept
- Founder wallet revenue routing
- Public API catalog and proof endpoint

Revenue is designed to stay founder-controlled first. Paid API revenue routes directly to the founder wallet, while the AI operator only keeps minimal gas for approved proof actions.
